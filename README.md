

## Проект 1: Тестирование бизнес-логики виртуального калькулятора
**Цель:** Разработка тестовой документации для гипотетического продукта на основе самостоятельно определённых требований и бизнес-логики.
**Описание:** Полный цикл работ от анализа предполагаемых требований до создания тест-плана и набора тест-кейсов. Проект демонстрирует умение мыслить системно и предугадывать сценарии использования.
**Артефакты:**
*   **Тест-план:** [`Test_Plan_Calculator`](ссылка_на_файл_в_вашем_репо) — документ, описывающий стратегию, объём, риски и подходы к тестированию.
*   **Тест-кейсы:** Серия проверок для операций базовых функций 

## Проект 2: Тестирование API в Postman (GoRest)
**Цель:** Освоение работы с API-запросами, настройкой окружения, переменных и автоматических проверок ответов сервера.
**Описание:** Практика с публичным API GoRest. Создание коллекции запросов, охватывающей жизненный цикл пользователь: создание (POST), чтение (GET), обновление (PATCH), удаление (DELETE). Настройка динамических переменных и написание скриптов для валидации ответов.
**Ключевые навыки и артефакты:**
*   Работа с **HTTP-методами** и **кодами ответов**.
*   Использование **переменных окружения** и **коллекции** в Postman.
*   Написание **скриптов на JavaScript** для пред- и постреквизитов, а также для автоматических проверок.
*   **Коллекция Postman:** [`GoRest_API_Tests`](ссылка_на_файл_в_вашем_репо) — готовая к запуску коллекция тестов.
